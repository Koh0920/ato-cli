# Trademark Guidelines

The names **Capsule** and associated logos are project marks used to identify this software.

## Allowed

- Descriptive use (e.g., “built with ato-cli”).
- Use in documentation and integrations that accurately refer to the project.
- Redistribution of unmodified official releases without implying endorsement.

## Not Allowed (without written permission)

- Using the project name or logo as part of your company/product name.
- Using the logo in marketing materials in a way that implies endorsement.
- Creating confusingly similar names, domains, or branding.

## Contact

For trademark permission requests, please open an issue in the relevant repository.
